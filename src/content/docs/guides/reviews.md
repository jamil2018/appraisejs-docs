AppraiseJS core includes a review flow for test-case quality control.

## Review capabilities

- Create reviews tied to test cases.
- Track reviewer assignments and statuses.
- Add comments and update review state.

## When to use

- Gate changes before large regression runs.
- Enforce test design standards across teams.
