## Step-by-step procedure

### Option 1: Scaffold a new project

```bash
npx create-appraisejs
cd <your-project-name>
npm run setup
npm run dev
```

### Option 2: Clone `appraisejs-core`

```bash
git clone https://github.com/jamil2018/appraisejs-core.git
cd appraisejs-core
npm run setup
npm run dev
```

After startup, open the local URL shown by the dev server.

## Checkpoint

- The local dev server starts without blocking setup errors.
- The AppraiseJS UI loads in the browser.
- You know whether to continue to [Quick Start](/getting-started/quick_start/) or jump to [Troubleshooting](/operations/troubleshooting/).

## Common mistakes

- Running `npm run dev` before `npm run setup` finishes.
- Treating browser dependency errors as AppraiseJS UI bugs instead of installation issues.
- Continuing into test authoring before confirming the app actually loads locally.

## Recovery steps

- Migration errors: run `npm run migrate-db`.
- Browser dependency errors: run `npm run install-playwright`.
- Stale or missing generated entities after setup: use `Settings -> Sync` after the UI loads, or run `npm run sync-all` if the UI is unavailable.
